# Security Policy

## Scope

This is a personal portfolio — a static site with no backend, no user accounts, and no sensitive data processing. The attack surface is intentionally minimal.

Areas in scope:
- Contact form (when implemented via Cloudflare Workers)
- Third-party dependencies with known vulnerabilities

Out of scope:
- The deployed static content itself
- Social engineering attempts

## Reporting a Vulnerability

If you find a security issue, please report it privately before disclosing it publicly.

**Contact:** diego.fdo.perez7@hotmail.com

Include in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I'll respond within 7 days and aim to resolve confirmed issues promptly.
